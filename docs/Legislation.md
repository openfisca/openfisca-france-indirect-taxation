La plupart des paramètres de la législation nécessaires pour simuler la fiscalité indirecte sont
regroupés dans le modèle. Ce sont des séries qui donnent, pour chaque année o`u il modifié, la
valeur d’un paramètre de la législation (ex: le taux normal de TVA, la TICPE sur le diesel ...).
Actuellement, le modèle gère la TVA, les taxes sur le tabac et les alcools, les assurances,
et les énergies. Côté préstation, le chèque énergie est égalment modélisé.
