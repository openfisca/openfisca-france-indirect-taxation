# Presentation

OpenFisca-France-Indirect-Taxation est un modèle de microsimulation de la taxation indirecte. Ce modèle vise à simuler l'ensemble des impôts indirects auxquels sont soumis les ménages en France. Le principe général d'un tel modèle est de simuler pour chaque individu d'une base de données représentative de la population française l'ensemble des taxes indirectes, sous la législation actuelle comme sous n'importe quelle autre législation afin d'en déduire les effets de réformes.

Le modèle procède comme suit : d'une part, il apparie statistiquement différentes bases de données permettant de représenter fidèlement la consommation des ménages et d'autre part il connecte la base ainsi produite à un calculateur socio-fiscal libre et collaboratif, [OpenFisca-France](https://openfisca.org/fr/).
